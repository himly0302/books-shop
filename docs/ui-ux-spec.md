# 免费图书分享小程序 UI/UX 设计规范

> 供 Gemini/Figma AI 生成设计稿

## 1. 设计风格

**风格定位**：清新现代 — 卡片式布局，色彩丰富，突出分类和操作按钮。适合内容浏览类工具产品。

**配色方案**：

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#4F6BF6` | 蓝紫色，主要按钮、链接、强调 |
| 主色悬停 | `#3B57D9` | 主色加深 |
| 辅助色 | `#FF8C3A` | 橙色，分类卡片装饰、标签 |
| 成功色 | `#00B894` | 绿色，收藏成功提示 |
| 警告色 | `#FDCB6E` | 黄色，提示信息 |
| 错误色 | `#FF6B6B` | 红色，加载失败 |
| 背景色 | `#F5F6FA` | 浅灰蓝，页面底色 |
| 卡片色 | `#FFFFFF` | 纯白，卡片/面板底色 |
| 文字主色 | `#1A1A2E` | 深色，标题和正文 |
| 文字次要 | `#8C8CA1` | 灰色，作者、简介、时间 |
| 边框色 | `#EEEEF3` | 极浅灰，分割线、卡片边框 |

**分类卡片配色（16 色）**：

每个分类分配独立色调，增加视觉区分度：
文学 `#6C5CE7`、历史 `#E17055`、科技 `#00B894`、学术 `#0984E3`、心理 `#E84393`、经济 `#FDCB6E`、社会 `#00CEC9`、哲学 `#A29BFE`、艺术 `#FF6B6B`、管理 `#55E6C1`、传记 `#F8A5C2`、教育 `#74B9FF`、其他 `#B2BEC3`、政治 `#FD79A8`、军事 `#636E72`、医学 `#81ECEC`

**圆角规范**：统一使用 `lg(12rpx)`，按钮使用 `full(48rpx)` 胶囊形

**阴影规范**：
- 卡片：`box-shadow: 0 2px 8px rgba(79, 107, 246, 0.08)`
- 弹窗：`box-shadow: 0 4px 20px rgba(0, 0, 0, 0.12)`

| H3 | 28rpx (14px) | Medium (500) | 卡片标题（书名） |
| 正文 | 28rpx (14px) | Regular (400) | 简介、详情文本 |
| 辅助 | 24rpx (12px) | Regular (400) | 作者、分类名、数量 |

## 3. 间距系统

基于 4px 基数：

- xs: 8rpx — 图标与文字间距
- sm: 16rpx — 表单项间距
- md: 24rpx — 卡片内 padding、段落间距
- lg: 32rpx — 页面左右 padding、区域间距
- xl: 48rpx — 大区域分割

## 4. 页面结构

### 首页

**布局**：单栏 + 卡片网格
**路径**：`/pages/index/index`

| 区域 | 内容 | 关键元素 |
|------|------|---------|
| 顶部 | 应用标题 + 搜索栏 | 搜索输入框（圆角胶囊形） |
| 最近浏览（P1） | 水平滚动的书籍封面缩略图 | 最近 10 条浏览记录，点击进入详情 |
| 分类网格 | 3 列卡片 Grid | 16 个分类卡片，每个显示分类名 + 数量 + 独立配色 |
| 书单推荐（P2） | 横向滑动书单卡片 | "必读经典"等精选书单 |

**关键交互**：
- 输入搜索词 → 防抖 300ms → 页面切换为搜索结果列表
- 点击分类卡片 → 跳转分类列表页
- 下拉刷新 → 重新加载 configs + index

### 分类列表页

**布局**：单栏列表
**路径**：`/pages/category/index?type=历史`

| 区域 | 内容 | 关键元素 |
|------|------|---------|
| 顶部 | 分类标题（含分类色块） + 返回按钮 | 导航栏 |
| 列表 | 书籍卡片纵向排列 | BookCard 组件 |
| 底部 | 加载更多提示 | "正在加载..." / "已全部加载" |

**关键交互**：
- 滚动到底部 → 自动加载下一页（每页 20 本）
- 下拉刷新 → 重置分页
- 点击书籍卡片 → 跳转详情页

### 书籍详情页

**布局**：单栏
**路径**：`/pages/detail/index?id=xxx`

| 区域 | 内容 | 关键元素 |
|------|------|---------|
| 顶部 | 返回按钮 + 操作栏 | 收藏按钮（心形）、分享按钮 |
| 封面 | 居中大图 | 400px 宽度，圆角 |
| 信息 | 书名（H1）、作者 · 分类标签 | 分类标签使用分类配色 |
| 简介 | 完整简介文本 | 正文样式 |
| 底部固定栏 | 操作按钮 | "复制下载链接"主按钮 |

**关键交互**：
- 点击"复制下载链接" → 复制到剪贴板 → Toast 提示
- 点击收藏 → 心形图标切换 → Toast "已收藏" / "已取消收藏"
- 点击分享 → 生成微信分享卡片

## 5. 组件规范

### BookCard（书籍卡片）

**变体**：列表模式（横向）、搜索结果模式
**尺寸**：高度自适应，固定 padding 24rpx
**内容**：左侧封面缩略图（120rpx × 160rpx） + 右侧信息（书名 2 行、作者 1 行、简介 2 行）
**圆角**：12rpx
**状态**：
- 默认 → 白色卡片
- 点击 → 背景色变为 `#F0F1FF`（主色极浅变体）

### CategoryCard（分类卡片）

**变体**：标准
**尺寸**：Grid 中等宽度（每行 3 个）
**内容**：分类色块背景 + 分类名 + 数量（如"190 本"）
**圆角**：12rpx
**状态**：
- 默认 → 分类色渐变背景
- 点击 → 轻微缩放（scale 0.97）

### SearchBar（搜索栏）

**变体**：标准
**尺寸**：全宽，高度 72rpx
**内容**：搜索图标 + 输入框 + 清除按钮（输入时显示）
**圆角**：36rpx（胶囊形）
**状态**：
- 默认 → 白色背景 `#FFFFFF`，边框 `#EEEEF3`
- 聚焦 → 边框变为主色 `#4F6BF6`
- 输入中 → 显示清除图标 ×

### 操作按钮

**变体**：Primary（主按钮，蓝紫实心）、Secondary（次按钮，白底蓝紫边框）
**尺寸**：高度 88rpx，全宽
**圆角**：44rpx（胶囊形）
**状态**：
- 默认 → 主色 `#4F6BF6`
- 点击 → 加深 `#3B57D9`
- 禁用 → `opacity: 0.5`

---

**通用状态规范**：

| 状态 | 实现方式 | 触发场景 |
|------|---------|---------|
| 加载中 | 骨架屏（Skeleton） | 页面首次加载 |
| 操作反馈 | Toast（2s 自动消失） | 复制成功、收藏成功 |
| 错误提示 | Toast + 重试按钮 | 网络异常、加载失败 |
| 空状态 | 插画 + "未找到相关书籍" + 搜索建议 | 搜索无结果 |

## 6. 响应式

仅小程序端，固定宽度设计（750rpx 设计稿），无需响应式断点。后续 H5 适配时再补充。

## 7. 动效

| 交互 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面切换 | 右滑入（小程序原生） | 300ms | ease-out |
| 分类卡片点击 | 缩放 0.97 → 1 | 100ms | ease |
| 收藏按钮 | 心形缩放弹跳 | 300ms | spring |
| 加载更多 | 底部旋转 Spinner | 循环 | linear |
